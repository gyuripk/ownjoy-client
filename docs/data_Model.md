### CCTV

데이터 관리 및 업데이트를 위해 필요한 항목

- **MNG_NO (관리번호)**: 데이터 중복을 방지하거나 특정 CCTV를 **고유하게 식별**할 때 필수
- **DAT_CRTR_YMD (데이터 기준일자)**: 데이터가 언제 만들어졌는지 확인
- **OPN_ATMY_GRP_CD**개방자치단체코드: 지자체 별로 묶어서 쿼리, 분석 등 용이

사용자에게 정보를 보여주기

- **INSTL_PRPS_SE_NM (설치목적)**: 이 CCTV가 왜 있는지 (예: 생활방범)
- **LCTN_ROAD_NM_ADDR (도로명주소)**: 사용자에게 익숙한 위치 정보
- **CAM_CNTOM (카메라대수)**: 시설의 규모를 알 수 있음
- **MNG_INST_NM (관리기관명)**: 문제가 생겼을 때 문의할 곳
- **MNG_INST_TELNO**관리기관전화번호

지도에 그리기

- **WGS84_LAT (위도)**: 세로 위치
- **WGS84_LOT (경도)**: 가로 위치
